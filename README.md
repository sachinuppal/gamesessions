# GameSessions — Multiplayer Game Session Manager

A real-time multiplayer game session management app — create rooms, invite friends, track scores, and manage game state across sessions.

## About

GameSessions makes it easy to organize casual multiplayer gaming with friends. No installs, no accounts — just create a session, share the link, and start playing. Built as a lightweight real-time companion for board games, card games, and casual mobile gaming meetups.

## Features

- Create and join game sessions with a shareable link
- - Real-time score tracking across players
  - - Session history and game logs
    - - Support for custom game types
      - - Mobile-first, works on any device
       
        - ## Tech Stack
       
        - - **Framework:** React + Vite
          - - **Language:** TypeScript
            - - **Styling:** Tailwind CSS
              - - **Real-time:** StackBlitz WebContainers
               
                - ## Local Development
